## Как использовать

### Автоматически (уже настроено)

Префаб уже добавлен на сцену и готов к работе. При запуске игры:
- Фон будет автоматически следовать за камерой
- Будет создан эффект бесконечного движения облаков

### Ручная настройка

Если вы хотите изменить параметры фона:

1. **В Inspector выберите `ParallaxBackground`**
2. **Настройте параметры:**
   - `Parallax Factor` (0-1):
     - 0 = фон не движется
     - 1 = фон движется с той же скоростью, что и камера
     - Рекомендуемое значение: 0.2-0.4 для эффекта глубины
   
   - `Background Sprite`: выберите другой спрайт фона при необходимости
   
   - `Sorting Order`: измените порядок сортировки, если фон перекрывается другими объектами
   
   - `Background Width/Height`: измените размеры фона под ваши нужды

## Как это работает

1. При старте скрипт создает две копии фона:
   - `Background_Top` - верхняя копия
   - `Background_Bottom` - нижняя копия (расположена под верхней)

2. В `LateUpdate()` фон движется за камерой с коэффициентом `parallaxFactor`

3. Когда одна из копий уходит из поля зрения камеры, она перемещается к другой копии, создавая бесконечный цикл

## Советы

- Для лучшего эффекта используйте изображения с прозрачными краями
- Можно добавить несколько слоев фона с разными коэффициентами параллакса для эффекта глубины
- Для горизонтального скроллинга измените код, чтобы двигать фон по оси X

## Устранение проблем

**Фон не виден:**
- Проверьте, что `Sorting Order` меньше, чем у других объектов (например, -10)
- Убедитесь, что Z-позиция фона больше, чем у камеры (камера на 0, фон на 10)

**Фон движется слишком быстро/медленно:**
- Измените `Parallax Factor` в Inspector

**Видны края фона:**
- Увеличьте `Background Width/Height`
- Или используйте изображение большего размера
